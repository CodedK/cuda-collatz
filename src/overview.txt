Implementations

gpu_full
GPU version with all optimizations. PTX and CUDA C

gpu_simple
Simplest possible GPU implementation, in both PTX and CUDA C

cpu_64bit_full
Full CPU version. Same optimizations as GPU and assembly core.

cpu_64bit_ref
Reference implementation that verifies the high level optimizations used in the GPU version in C++

cpu_simple
Simplest possible CPU implementation in C++

mksieve_64bit
Generate the sieve table for sieve optimization
